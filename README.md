## ✨ Features

- **Responsive Design** — works on mobile, tablet, and desktop
- **Smooth Animations** — scroll-triggered fade-ins and staggered entrance effects
- **Sticky Navbar** — transparent on load, frosted-glass effect on scroll
- **Mobile Hamburger Menu** — animated open/close with accessible markup
- **Project Cards** — hover overlays with direct links to project pages
- **Skills Section** — interactive pill tags with hover effects
- **Subtle Cursor Trail** — accent dot that follows the mouse in the hero section

## 🏃 Getting Started

No build tools or dependencies required. Just open the files in a browser.

```bash
# Clone the repository
git clone https://github.com/aditshah29/portfolio.git

# Open in browser
open index.html
```

Or simply drag `index.html` into any modern browser.

---

## 🎨 Design Decisions

- **Color palette:** Dark background (`#0b0b0f`) with a lime-green accent (`#c8f064`) for contrast and personality
- **Typography:** DM Serif Display (headings) paired with DM Sans (body) for a modern editorial feel
- **Animations:** CSS keyframes for hero entrance; `IntersectionObserver` for scroll-triggered reveals — no external animation libraries needed
